Required admin editing features:

1. Editable core booking details:

* date
* time / time window
* duration / hours
* van size
* help option
* pickup address
* pickup stairs / floor
* drop-off address
* drop-off stairs / floor
* notes
* payment status

2. Extra Stops:
   Add a button: “+ Add Extra Stop”

Each extra stop should have:

* address
* charge
* notes

If no extra stop is added, do not show it in Telegram or customer messages.

3. Extra charges:
   Add optional editable charges:

* extra time
* stairs charge
* congestion charge
* outside M25 charge
* manual extra charge

Each charge should have:

* label
* amount
* notes

4. Price breakdown:
   Admin panel must clearly show:

* base price
* duration
* hourly rate
* extra stops total
* extra time total
* stairs charge
* congestion charge
* outside M25 charge
* manual charges
* agreed total price
* deposit paid
* remaining balance

   If deposit is already paid, it must stay fixed unless I manually edit it.

Example:
Original total: £180
Deposit paid: £54

Updated total: £215

Deposit paid must stay £54.
Remaining balance must automatically become £161.

Formula:
remaining balance = agreed total price - deposit paid

   Add two separate admin actions:

A) “Save Changes”

* Saves changes in admin panel
* Updates Google Sheets
* Does NOT send Telegram update
* Does NOT send customer message

B) “Save & Confirm Update”

* Saves changes
* Updates Google Sheets
