* Service type
* Van size
* Help option
* Time slot / time window

Van size:

* Small Van
* Medium Van
* Large Van
* Luton Van

Help option:

* No help needed
* Driver help
* Driver + Helper

Time slot:

* Custom time

### 2. Admin pricing should use the same pricing engine as the customer booking form

The admin panel should use exactly the same pricing logic as the website.

Please include the existing pricing rules:

* Congestion Charge (£18 per booking)
* Outside M25 (£1 per mile)
* Stairs (£10 per floor)
* Additional Stop (£5 per stop)
* Extra Time (using the existing pricing logic)

The admin should not need to remember any prices manually.

For every pricing option:

* Enable/disable toggle or checkbox
* Automatic calculation using the existing pricing engine
* Manual override when required

Examples:

* Additional Stop → admin adds a stop → automatically adds £5 (editable if needed)
* Stairs → admin enters number of floors → automatically calculates £10 × floors
* Outside M25 → use the existing mileage calculation where available, otherwise allow manual mileage entry
* Congestion Charge → simple on/off toggle
* Extra Time → select 30, 60, 90 minutes etc. using the existing pricing logic already used by the booking form
